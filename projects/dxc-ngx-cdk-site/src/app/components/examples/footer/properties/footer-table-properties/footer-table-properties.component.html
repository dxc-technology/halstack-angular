Name Default Description logoSrc: string The path of an icon to replace the theme logo. socialLinks: object[] [] An array of objects representing the links that will be rendered as icons at the top-right side of the footer. Each object has the following properties:
  • logoSrc: The path of an icon for the link.
  • href: URL of the page the link goes to.
bottomLinks: object[] [] An array of objects representing the links that will be rendered at the bottom part of the footer. Each object has the following properties:
  • text: Text for the link.
  • href: URL of the page the link goes to.

You can pass an object with 'top', 'bottom', 'left' and 'right' properties in order to specify different padding sizes. tabIndexValue: number 0 Value of the tabindex for all interactuable elements, except those inside the custom area.
