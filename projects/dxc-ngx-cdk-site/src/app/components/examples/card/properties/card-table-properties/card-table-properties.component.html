Name Default Description imageSrc: string URL of the image that will be placed in the card component. imageBgColor: string 'black' Color of the image background. imagePadding: any (string | object) Size of the padding to be applied to the image section of the component ('xxsmall' | 'xsmall' | 'small' | 'medium' | 'large' | 'xlarge' | 'xxlarge'). You can pass an object with 'top', 'bottom', 'left' and 'right' properties in order to specify different padding sizes. imagePosition: string ('after' | 'before') 'before' Whether the image should appear in relation to the content. contentPadding: any (string | object) Size of the padding to be applied to the content section of the component ('xxsmall' | 'xsmall' | 'small' | 'medium' | 'large' | 'xlarge' | 'xxlarge'). You can pass an object with 'top', 'bottom', 'left' and 'right' properties in order to specify different padding sizes. linkHref: string If defined, the card will be displayed as an anchor, using this prop as "href". Component will show some visual feedback on hover. onClick: EventEmitter This event will emit when the user clicks the card. Component will show some visual feedback on hover. imageCover: boolean false Whether the image must cover the whole image area of the card. margin: any (string | object) Size of the margin to be applied to the component ('xxsmall' | 'xsmall' | 'small' | 'medium' | 'large' | 'xlarge' | 'xxlarge'). You can pass an object with 'top', 'bottom', 'left' and 'right' properties in order to specify different margin sizes. tabIndexValue: number 0 Value of the tabindex given when there is an href. outlined: boolean true Whether the card must be outlined.
