DXC Angular Components is a library made for Angular 13 applications. In order to use it, you need to include it in an existing Angular project.

If you are creating a Angular application from scratch, we recommend to follow the next steps to install from scratch.

DXC Angular Components is distributed as an npm package. To use it in an existing project:

It has defined a peerDependencies in the package.json. As you know, this section of depedencies are not installed automatically by npm. System will warn you if you run npm install and it does not find this dependency.
