Properties

Name Default Description mode: string ('default' | 'alternative') 'default' Mode for the color of the accordion. theme: string ('light' |'dark') 'light' Uses one of the available component themes. label: string Text to be placed next to the button. iconSrc: string URL of the icon that will be placed next to the button label. iconPosition: 'before' | 'after' Whether the icon should appear after or before the label. assistiveText: string Assistive text to be placed on the right side of the panel. disabled: boolean false If true, the component will be disabled. isExpanded: boolean false If true, the component will be expanded. onClick: EventEmitter This function will be called when the user clicks the icon to open/close the panel. The state of the panel (opened/closed) should be passed as a parameter margin: any (string | object) Size of the margin to be applied to the component ('xxsmall' | 'xsmall' | 'small' | 'medium' | 'large' | 'xlarge' | 'xxlarge'). You can pass an object with 'top', 'bottom', 'left' and 'right' properties in order to specify different margin sizes. padding: any (string | object) Size of the padding to be applied to the custom area ('xxsmall' | 'xsmall' | 'small' | 'medium' | 'large' | 'xlarge' | 'xxlarge'). You can pass an object with 'top', 'bottom', 'left' and 'right' properties in order to specify different padding sizes.
