Name Default Description label: string Text to be placed inside the spinner. mode: string ('large' | 'small' | 'overlay') 'large' Available modes of the spinner. value: string The value of the progress indicator. If it's received the component is determinate, otherwise is indeterminate. showValue: boolean false If true, the value is displayed inside the spinner. margin: any (string | object) Size of the margin to be applied to the component ('xxsmall' | 'xsmall' | 'small' | 'medium' | 'large' | 'xlarge' | 'xxlarge'). You can pass an object with 'top', 'bottom', 'left' and 'right' properties in order to specify different margin sizes.
