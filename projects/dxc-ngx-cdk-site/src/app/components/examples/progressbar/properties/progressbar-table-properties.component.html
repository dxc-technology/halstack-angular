Name Default Description label: string Text to be placed above the progress bar. helperText: string Helper text to be placed under the progress bar. overlay: boolean true If true, the progress bar will be displayed as a modal. value: string The value of the progress indicator. If it's received the component is determinate otherwise is indeterminate. showValue: boolean false If true, the value is displayed above the progress bar. margin: string | object Size of the margin to be applied to the component ('xxsmall' | 'xsmall' | 'small' | 'medium' | 'large' | 'xlarge' | 'xxlarge'). You can pass an object with 'top', 'bottom', 'left' and 'right' properties in order to specify different margin sizes.
