Name Default Description mode: 'horizontal' | 'vertical' 'horizontal' The wizard can be showed in horizontal or vertical. currentStep: number 0 Defines which step is marked as the current. The numeration starts in 0. defaultCurrentStep: number Initially selected step, only when it is uncontrolled. tabIndexValue: number 0 Value of the tabindex attribute that is given to all the steps. onStepClick: EventEmitter This event will emit in case the user clicks a step. The step number will be passed as a parameter. margin: string | object Size of the margin to be applied to the component ('xxsmall' | 'xsmall' | 'small' | 'medium' | 'large' | 'xlarge' | 'xxlarge'). You can pass an object with 'top', 'bottom', 'left' and 'right' properties in order to specify different margin sizes.
