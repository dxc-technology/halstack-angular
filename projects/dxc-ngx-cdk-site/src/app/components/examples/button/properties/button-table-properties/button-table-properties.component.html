URL of the icon that will be placed next to the button label. iconPosition: 'before' | 'after' 'before' Whether the icon should appear after or before the label. disabled: boolean false If true, the component will be disabled. onClick: EventEmitter This function will be called when the user clicks the button. The event object will be passed as a parameter. margin: string | object Size of the margin to be applied to the component ('xxsmall' | 'xsmall' | 'small' | 'medium' | 'large' | 'xlarge' | 'xxlarge'). You can pass an object with 'top', 'bottom', 'left' and 'right' properties in order to specify different margin sizes. size: string 'fitContent' Size of the component ('small' | 'medium' | 'large' | 'fillParent' | 'fitContent'). tabIndexValue: number 0 Value of the tabindex.
