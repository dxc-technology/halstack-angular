Tab Properties

Name Default Description label: string Text to be placed within the tab. iconSrc: string Deprecated. The path of an icon to be placed within the tab. disabled: boolean false Whether the tab is disabled. notificationNumber: boolean | number It can have boolean type or number type. If the value is 'true', an empty badge will appear. If it is 'false', no badge will appear.
