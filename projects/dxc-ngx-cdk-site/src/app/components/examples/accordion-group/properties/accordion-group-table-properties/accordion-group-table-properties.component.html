Name Default Description indexActive: number The index of the active accordion. If undefined, the component will be uncontrolled and the active accordion will be managed internally by the component. If null, the component will be controlled and all accordions will be closed. disabled: boolean false If true, the component will be disabled. onActiveChange: EventEmitter This event will emit in case the user clicks on an accordion. The index of the clicked accordion will be passed as a parameter. margin: string | object Size of the margin to be applied to the component ('xxsmall' | 'xsmall' | 'small' | 'medium' | 'large' | 'xlarge' | 'xxlarge'). You can pass an object with 'top', 'bottom', 'left' and 'right' properties in order to specify different margin sizes.
