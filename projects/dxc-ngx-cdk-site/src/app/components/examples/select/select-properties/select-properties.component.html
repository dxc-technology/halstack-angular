Name Default Description label: string Text to be placed above the select. name: string Name attribute of the input element. value: string | string[] Value of the select. If undefined, the component will be uncontrolled and the value will be managed internally by the component. defaultValue: string | string[] Default value given to the select and also maintains the uncontrolled behaviour. options: Option[] | OptionGroup[] List of options. The following interfaces should be imported.
Option:
  • Label: string: Label of the option to be shown in the select.
  • Value: string: Value of the option. It must be unique.
  • Icon: string: Icon of the option. It is optional.
OptionGroup:
  • Label: string: Label of the group to be shown in the select.
  • Options: Option[]: List of options of the group.
helperText: string Helper text to be placed above the select. placeholder: string Placeholder of the select. searchable: boolean false If true, an input will appear to filter the list of options. multiple: boolean false If true, multiple options can be selected. If false, only a single option can be selected. disabled: boolean false If true, the component will be disabled. optional: boolean false If true, the select will be optional, showing (Optional) next to the label. error: string If it is defined, the component will change its appearance, showing the error below the select component. onChange: EventEmitter This function will be called when the user select an option. An object including the new value (or values) and the error (if the value selected is not valid) will be passed to this function. An example of this object is: {{ "{" }} value: value, error: error{{ "}" }}. If there is no error, error will be null. onBlur: EventEmitter This function will be called when the select loses the focus. An object including the value (or values) and the error (if the value selected is not valid) will be passed to this function. An example of this object is: {{ "{" }} value: value, error: error {{ "}" }}. If there is no error, error will be null. margin: string | object Size of the margin to be applied to the component ('xxsmall' | 'xsmall' | 'small' | 'medium' | 'large' | 'xlarge' | 'xxlarge'). You can pass an object with 'top', 'bottom', 'left' and 'right' properties in order to specify different margin sizes. size: string | object 'medium' Size of the component ('small' | 'medium' | 'large' | 'fillParent'). tabIndexValue: number 0 Value of the tabindex attribute.
