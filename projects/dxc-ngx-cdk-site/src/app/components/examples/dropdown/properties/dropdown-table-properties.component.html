Properties

Name Default Description options: object[] [] An array of objects representing the selectable options. Each object has the following properties:
  • label: Option display value
  • value: Option value
  • iconSrc: URL of the icon that will be placed next to the option label (Optional)
optionsIconPosition: 'before' | 'after' 'before' In case options include icons, whether the icon should appear after or before the label. iconSrc: string URL of the icon that will be placed next to the dropdown label. iconPosition: 'before' | 'after' 'before' Whether the icon should appear after or before the label. label: string Text to be placed when the list of options is not displayed. theme: string ('light' | 'dark') 'light' Uses one of the available component themes. mode: 'basic' | 'outlined' 'basic' Uses one of the available component modes. name: string Name attribute of the input element. disableRipple: boolean false If true, the ripple effect will be disabled. showCaret: boolean true Whether the arrow next to the label is displayed or not. selectOption: EventEmitter This event will be triggered when the selection changes. The string with the key of the selected value will be passed as a parameter. margin: any (string | object) Size of the margin to be applied to the component ('xxsmall' | 'xsmall' | 'small' | 'medium' | 'large' | 'xlarge' | 'xxlarge'). You can pass an object with 'top', 'bottom', 'left' and 'right' properties in order to specify different margin sizes. size: any (string | object) 'fitContent' Size of the component ('large' | 'fillParent' | 'fitContent').
