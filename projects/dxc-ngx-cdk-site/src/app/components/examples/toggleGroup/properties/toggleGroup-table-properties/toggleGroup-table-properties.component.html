Name Default Description label: string Text to be placed next to the input. helperText: string Assistive text to be placed bellow the input. value: string | string[] Value(s) of the toggle(s) that are toggled. If undefined, the component will be uncontrolled and the value will be managed internally by the component. multiple: boolean false If true, the toggle group will support multiple selection. In that case, value must be an array of numbers with the keys of the selected values. disabled: boolean false If true, the component will be disabled. tabIndex: number Value of the tabindex which its propagated to their children components. onChange: EventEmitter This function will be called when the user changes the state of any toggle. The new value or values will be passed as a parameter. margin: string | object Size of the margin to be applied to the component ('xxsmall' | 'xsmall' | 'small' | 'medium' | 'large' | 'xlarge' | 'xxlarge'). You can pass an object with 'top', 'bottom', 'left' and 'right' properties in order to specify different padding sizes.
