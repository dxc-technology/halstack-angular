Name Default Description level: number 1 Defines the heading level from 1 to 5. The styles of the heading are applied according to the level. text: string Heading text. weight: 'light' | 'normal' | 'bold' Modifies the default weight of the heading. margin: string | object Size of the margin to be applied to the component ('xxsmall' | 'xsmall' | 'small' | 'medium' | 'large' | 'xlarge' | 'xxlarge'). You can pass an object with 'top', 'bottom', 'left' and 'right' properties in order to specify different margin sizes.
